## Covered

- Styled RangeCalendar is live on both React and Solid comparison stacks.
- Interactive controls cover controlled range value, focusedValue, visible month count, page behavior, selection alignment, first day of week, min/max range constraints, unavailable dates, non-contiguous ranges, disabled, read-only, invalid, and error text.
- Solid RangeCalendar now uses S2 header/nav icons, title typography, month-grid composition, range fill/border treatment, unavailable strike, and invalid help text.
- Headless RangeCalendar exposes cell wrapper styling and first/last row/column render props needed to avoid inferred calendar geometry.
- Seven-column rows and deterministic `224px` month grid geometry are asserted in package and comparison browser tests.
- DateRangePicker reuses RangeCalendar without the old fixed `272px` override that could create a trailing blank gutter.
